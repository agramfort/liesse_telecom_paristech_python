Introduction¶

  • numpy est un module utilis dans presque tous les projets de calcul numrique sous Python
    • Il fournit des structures de donnes performantes pour la manipulation de vecteurs, matrices et tenseurs plus gnraux
    • numpy est crit en C et en Fortran d'o ses performances leves lorsque les calculs sont vectoriss (formuls comme des oprations sur des vecteurs/matrices)
  • matplotlib est un module performant pour la gnration de graphiques en 2D et 3D
    • syntaxe trs proche de celle de Matlab
    • supporte texte et tiquettes en $\LaTeX$
    • sortie de qualit dans divers formats (PNG, PDF, SV, EPS...)
    • interface graphique intractive pour explorer les figures

Pour utiliser numpy et matplotlib il faut commencer par les importer :

In [2]:
import numpy as np
import matplotlib.pyplot as plt

On peut plus simplement faire :

In [3]:
from numpy import *
from matplotlib.pyplot import *

Arrays numpy¶

Dans la terminologie numpy, vecteurs, matrices et autres tenseurs sont appels arrays.

Création d'arrays numpy¶

Plusieurs possibilits:

  • a partir de listes ou n-uplets Python
  • en utilisant des fonctions ddies, telles que arange, linspace, etc.
  • par chargement partir de fichiers

Slicing ou accès par tranches¶

Slicing fait rfrence la syntaxe M[start:stop:step] pour extraire une partie d'un array :

In [78]:

A
Out[78]:
array([1, 2, 3, 4, 5])
In [79]:
A[1:3]
Out[79]:
array([2, 3])

Les tranches sont modifiables :

In [80]:
A[1:3] = [-2,-3]
A
Out[80]:
array([ 1, -2, -3,  4,  5])

On peut omettre n'importe lequel des argument dans M[start:stop:step]:

In [81]:

Out[81]:

Out[83]:
array([ 1, -2, -3])
In [84]:

Copy et "deep copy"¶

Pour des raisons de performance Python ne copie pas automatiquement les objets (par exemple passage par rfrence des paramtres de fonctions).

In [163]:
A = np.array([[0,  2],[ 3,  4]])
A
Out[163]:
array([[0, 2],
       [3, 4]])
In [164]:
B = A
In [165]:
# changer B affecte A
B[0,0] = 10
B
Out[165]:
array([[10,  2],
       [ 3,  4]])
In [166]:
A
Out[166]:
array([[10,  2],
       [ 3,  4]])
In [167]:
B = A
print B is A

True

Pour viter ce comportement, on peut demander une copie profonde (deep copy) de A dans B

In [168]:
# B = np.copy(A)
B = A.copy()
In [169]:
# maintenant en modifiant B, A n'est plus affect
B[0,0] = -5

B
Out[169]:
array([[-5,  2],
       [ 3,  4]])
In [170]:
A  # A est aussi modifi !
Out[170]:
array([[10,  2],
       [ 3,  4]])
In [171]:
print(A - A[:,0])  # FAUX
print(A - A[:,0].reshape((2, 1)))  # OK

[[ 0 -1]
 [-7  1]]
[[ 0 -8]
 [ 0  1]]
